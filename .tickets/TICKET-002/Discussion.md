# Discussion

## Decisions

### Why Vite?
- Fast development server with HMR
- Fast production builds
- Modern tooling
- Simple configuration

### Why TypeScript?
- Type safety
- Better IDE support
- Self-documenting code
- Catches errors early

### Project Structure
- Components in `src/components/`
- Services in `src/services/`
- Stores in `src/stores/`
- Types in `src/types/`
- Utils in `src/utils/`
