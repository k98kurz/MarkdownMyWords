# Discussion

## Theme Implementation

### CSS Variables
- Use CSS variables for colors
- Switch via data attribute
- Easy to maintain
- No JavaScript needed for styling

### System Preference
- Detect prefers-color-scheme
- Default to system preference
- Allow manual override
- Persist user choice

## Color Palette

### Light Theme
- White background
- Dark text
- Subtle borders
- Blue accent

### Dark Theme
- Dark background
- Light text
- Subtle borders
- Light blue accent

## Persistence

- Store in GunDB user settings
- Load on app initialization
- Sync across devices
- Fallback to system preference
