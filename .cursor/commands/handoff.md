# /handoff [TICKET-XXX]

Generate handoff notes for a ticket.

## Usage
```
/handoff TICKET-001
```

## Parameters
- **ticketId**: The ticket ID to generate handoff for

## Output
Generates structured handoff notes including:
- Current status and progress
- What was completed
- What remains to be done
- Key decisions made
- Blockers or concerns
- Next steps

## Notes
Handoff notes are appended to Discussion.md
