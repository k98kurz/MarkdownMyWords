# /ticket status

Show all active tickets and their status.

## Usage
```
/ticket status
```

## Output
Lists all tickets that are:
- in_progress
- pending_review
- pending_completion_review

## Example Output
```
Active Tickets:
- TICKET-009: Integrate Ticket Scripts (in_progress) - 45% complete
- TICKET-008: Improve Links (pending_review)
```
